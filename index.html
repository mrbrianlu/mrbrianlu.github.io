  • Ecommerce Shop

    Spring 2019 View Source

    Technologies Used

    • PHP
    • HTML 5
    • CSS 3
    • Bootstrap
    • MySQL

    Project

    An ecommerce website that sells LA sport teams apparel, with ability to let user add items into shopping cart and check out with user’s information, then send email for confirmation. Back-end is built using PHP while using MySQL as Database.

    Project Difficulties

    Product page needs to display product information dynamically. Default set at 8 items per page. User has the ability to sort by name or price. And, user's shopping cart will remain in session even after reopen window.

    My Solution

    By using "LIMIT" and "ORDER BY" in SQL statement, I was able to create pagination amd sorting function. For cart function, server stores "cart" array in user session, and set lifetime of the cookie for two weeks.

    Notable Features

    • MVC design pattern
    • Using Prepared statement to prevent SQL injection
    • Pagination shows 8 products per page
    • Sorting products by name or price
    • Save user shopping cart in session cookie
    • Edit quantity in shopping cart and calculate total price

  • Pizza Ordering App

    Spring 2019 View Source

    Technologies Used

    • Java
    • XML
    • SQLite

    Project

    A pizza ordering app that let users choose size and toppings of pizza. It will also collect user information and total price in SQLite database.

    Project Difficulties

    My first mobile app development. Mobile app has different UX/UI strategy then webiste or desktop software due to it's limited screen size. It is best practice to use single window or page for each function of the app.

    My Solution

    In Android development, a page is typically represent by one Activity. This app uses four Activities that each handle different jobs, such as welcome screen, collect order info, and confirmation page, etc. There is also one Fragment that let user customize pizza.

    Notable Features

    • User has choice of different pizza size
    • Adding each topping will be shown on the pizza display
    • Collect user data and store on SQLite
